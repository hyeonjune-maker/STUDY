HTML공부

 

태그 사용 통계

  1. head : 웹 페이지의 정보, 문서에서 사용할 외부 파일들을 링크할 때 사용
  2. body: 브라우저에 실제 표시되는 내용
  3. html : 웹페이지의 시작과 끝
  4. title : 문서 제목
  5. meta : 문자 인코딩 및 문서 키워드, 요약 정보
  6. div : 아무런 의미 X, 컨텐츠들을 어떤 목적에 따라 묶어야 할 때 사용한다.
  7. a : anchor, 웹 페이지나 외부 사이트 연결
  8. script : 코드 삽입
  9. link : 외부 파일을 연결할 때 사용한다.
  10. p : 단락
  11. span : 아무런 의미 X, 컨텐츠들을 어떤 목적에 따라 묶어야 할 때 사용한다.
    div는 줄바꿈O, span은 줄바꿈X
  12. li : ul, ol 안에 들어가는 항목을 나열할 때 사용하는 태그
  13. ul : unordered list, 순서가 없는 list
  14. br : 줄
    바꿈
  15. style  : 스타일 정보를 정의할 때 사용하는 태그

  16. h1

    h2

    h3

  17. input : form의 요소중 하나. 사용자가 정보를 입력하는 부분을 만들어야 할 때 사용한다.
  18. form : form 생성
  19. strong : 중요한 내용 강조
  20. table : 표를 만드는 태그
  21. tr : table row 약자로, 가로줄을 만드는 역할 (기본값은 보통 글씨체에, 왼쪽 정렬)
  22. td: table data 약자로, 셀을 만드는 역할 (기본값은 보통 글씨체에, 왼쪽 정렬)
태그 내용
tr table row 약자로, 가로줄을 만드는 역할 (기본값은 보통 글씨체에, 왼쪽 정렬)
td table data 약자로, 셀을 만드는 역할 (기본값은 보통 글씨체에, 왼쪽 정렬)
